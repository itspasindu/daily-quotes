Python Cheat Sheet

Print Statement: print("Hello, World!")

Comments: # This is a comment

Integer: x = 5

Float: y = 3.14

String: name = "Python"

List: numbers = [1, 2, 3]

Tuple: coordinates = (10, 20)

Dictionary: person = {"name": "Alice", "age": 25}

Set: unique_numbers = {1, 2, 3}

If Statement: if x > 0: print("Positive")

For Loop: for i in range(5): print(i)

While Loop: while x < 5: x += 1

Defining a Function: def my_function(): pass

Function with Parameters: def add(a, b): return a + b

Lambda Function: add = lambda a, b: a + b

Defining a Class: class MyClass: pass

Constructor: def __init__(self, param): self.param = param

Creating an Object: obj = MyClass(param)

Importing a Module: import module_name

Importing Specific Function: from module_name import function_name

Opening a File: file = open("filename.txt", "r")

Reading a File: content = file.read()

Writing to a File: file.write("Hello, World!")

Closing a File: file.close()

Try-Except Block: try: pass except Exception as e: pass

Finally Block: finally: pass

Basic Syntax: [expression for item in iterable if condition]

Example: squares = [x**2 for x in range(10)]

Defining a Decorator: def decorator_function(original_function): pass

Applying a Decorator: @decorator_function

Defining a Generator: def generator_function(): yield value

Using a Generator: for value in generator_function(): pass
